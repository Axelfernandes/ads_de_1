# Ad Analytics Dashboard

Ad Campaign Analytics Dashboard showcasing SQL, React, and AWS skills.

## Tech Stack
- **Frontend**: React + TypeScript + Vite
- **Charts**: Recharts
- **Icons**: Lucide React
- **Backend (planned)**: AWS Amplify, Lambda, S3, DynamoDB, Athena

## Features
- Campaign Performance Dashboard
- Merchant Analytics
- A/B Test Results
- Data Upload to S3

## Getting Started

```bash
npm install
npm run dev
```

## AWS Architecture (Planned)
- S3: Raw data lake
- DynamoDB: Campaign/merchant data
- Athena: SQL analytics
- Lambda: Data processing
- Amplify: Hosting
